## MemoBox

Projeto simples de um sistema de CRUD de notas online.
- Organização de notas em pastas;
- Formatação de texto.

### Futuras features
- Autenticação e autorização com JWT e bcrypt;
- Configurações de conta e dados do site;
- Cores de pastas;
- Templates de pastas;
- Edição de notas.

### Tecnologias usadas
- Node.js/Express
- Axios
- Sequelize
- MySQL
- Dotenv
- CORS
